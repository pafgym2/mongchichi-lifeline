# 몽치치패밀리 라이프라인

가족의 중요한 순간을 날짜순 타임라인으로 기록하는 웹앱입니다.

## 온라인 연결 순서

1. Supabase 프로젝트를 만듭니다.
2. Supabase SQL Editor에서 `supabase-schema.sql` 내용을 실행합니다.
3. Supabase Project Settings > API에서 Project URL과 anon public key 또는 publishable key를 복사합니다.
4. `config.js`에 값을 넣습니다.
5. Vercel에 이 폴더를 배포합니다.

배포가 끝나면 Vercel 주소를 가족에게 보내면 됩니다.
